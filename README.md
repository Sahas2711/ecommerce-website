# E-Commerce Website

This is an E-commerce platform built using Django for the backend and Bootstrap for the frontend. The platform allows users to browse products, sign up, log in, and manage their accounts. It includes a dark-themed interface with a user-friendly layout.

## Features
- **User Authentication**: Sign up, login, and logout functionality using Django's built-in authentication system.
- **Product Browsing**: Users can view a list of products, categorized for easy navigation.
- **Responsive Design**: The website is fully responsive and mobile-friendly, ensuring a seamless experience across all devices.
- **Dark Mode**: The entire website is styled with a dark theme for a modern look.
- **Search Functionality**: A search bar is available for users to search for products.

## Tech Stack
- **Backend**: Django
- **Frontend**: HTML, CSS, Bootstrap 5
- **Database**: SQLite (default with Django, can be switched to other databases like PostgreSQL)
- **Version Control**: Git
